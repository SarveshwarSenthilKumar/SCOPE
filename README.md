## Quick Start

### Viewing the Documentation

Open the documentation directly in your browser:

```bash
# Simply open the HTML file in your browser
open index.html
# or double-click the file in your file explorer
```

### Running the Live SCOPE Application

```bash
# Start the main SCOPE application
python app.py
```

The SCOPE web application will be available at: `http://localhost:5000`

## Technical Stack

- Backend: Python 3.11+, Flask 2.3.3
- Frontend: HTML5, CSS3, JavaScript, Bootstrap 5.3.0
- Database: SQLite3 with optimized indexing
- AI/ML: Google Gemini AI, scikit-learn, numpy
- Real-time: Server-Sent Events (SSE)
- Hardware: Raspberry Pi, I2C/UART sensors, mmWave radar
- Notifications: SMTP (Gmail), Microsoft Teams Webhooks
- PDF Generation: WeasyPrint, Matplotlib
